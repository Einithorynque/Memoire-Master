## Feature Analysis 

### Définition
- Les auteurs·ices considèrent comme [[Fonctionnalités|fonctionnalités]] les fonctions d'une application que les utilisateurs·ices exécutent, incluant tous celles auxquelles iels peuvent accéder, qu'iels peuvent modifier, contrôler, ou qu'iels n'utilisent pas mais connaissent surement.
  >« A feature can be any function that the user executes in the app, including anything the user can access, modify, or control, as well as functions performed on the user’s behalf » (Hasinoff et Bivens, 2021, p. 97)
- Les **fonctions cachés** aux utilisateurs·ices ne sont donc **pas comptées** comme [[Fonctionnalités]]. (cf : cookies)
