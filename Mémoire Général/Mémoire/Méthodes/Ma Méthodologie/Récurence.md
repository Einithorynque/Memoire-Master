Pour compter la récurrence :
1. La récurrence est compté dans le total de toutes les pages explorées du site
2. Elle est compté de la manière suivante :
	- Si une fonctionnalité est dans le header, noter 'header' dans le compte de récurrence et faire les compte des autres formes d'apparition ci dessous.  
	- Si dans le corps de la page un lien renvoie à une fonctionnalité précise sur une autre page : +1
		- Alors, la fonctionnalité sur sa propre page compte pour : +0
	- Sinon si la fonctionnalité apparait sur une page sans être annoncé : +1 

---

La récurrence d'une [[Fonctionnalités|fonctionnalité]] d'un site est calculée par le nombre de fois qu'elle apparait et qu'il est considéré qu'elle [[Demander|demande]] une action. Ce compte est effectué sur chaque page du site dont le lien est présent, soit sur sa page d'accueil, soit sur son haut de page, excluant donc les pages uniquement présentent sur les plans de sites. La présence d'une fonctionnalité dans le haut de page n'est compté qu'une fois même si répétée à chaque page.

Elle prend la forme : récurrence/nombre de page étudiée

Décision sur une ambiguïté :
- [[Les écologistes]], la fonctionnalité [[Télécharger l'application Ecolo!]] a été comptée sur chaque page même si elle est située dans la balise footer du site car 1. celle ci se démarque visuellement du footer, 2. cette forme de rappel dans le footer, aussi démarqué, est anodin.
