[Sustainable Web Manifesto]([https://www.sustainablewebmanifesto.com/](https://www.sustainablewebmanifesto.com/)  )
## Eco interactions :
- Un site web open source, dont on voit les couture peut se ressentir
- Un site web dont dont le rapport homme machine est honnête, sans tromperie ou exploitation 
- Un site web se ressentant comme résilient : fini temporellement et géographiquement en fonction d'un besoin

## Résumé :
Le web si pays = 4ème pollueur  
### Manifeste pour un :  
> **1. Web propre**
> Qui marche à l’énergie renouvelable  
> **2. Web efficient**
> Qui consomme le moins de ressources possibles
> **3. Web ouvert**
> Open source  
> **4. Web honnête**
>  Le design n’y sera pas utiliser pour tromper et exploiter les utilisateurs  
> **5. Web régénératif**
> Qui supporte une économie luttant contre les inégalités et pour la planète
> **6. Web résilient**
> Les services fonctionneront uniquement dans des lieux et sur une durée pour lesquels ils répondront à un réel besoin
